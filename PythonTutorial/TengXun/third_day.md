## 面向对象中常用术语：

- 类：可以理解是一个模板，通过它可以创建出无数个具体实例。
- 对象：实例化后的类称为对象。
- 属性：类中的所有变量和方法称为属性。
- 方法：类中的所有函数通常称为方法。


## 类的定义
Python 中定义一个类使用 class 关键字实现，其基本语法格式如下：

```
class 类名：
    多个（≥0）类属性...
    多个（≥0）类方法...
```
